WIP: Design-based vs Model-based framework

2020-07-20

The postulate of randomness thus resolves itself into the question, “Of what population is this a random sample?” which must frequently be asked by every practical statistician. (Fisher 1922)

In Fisher’s model-based framework, you begin with a hypothetical infinite population that represents all possible \(y\)’s and assume that your data set is a simple random sample (SRS) from this infinite population. If your data set is NOT a SRS from your hypothetical population, then your hypothetical population is incorrect. In this sense, the model-based framework does not require an actual random sample; it only requires that the distributional assumptions of the infinite population match that of the sample.

In constrast, Neyman’s design-based framework does not construct a model or a hypothetical infinite population. In the design-based framework the targets of inference are not the hypothetical infinite population parameters, but to estimate the parameters of a fixed, finite population e.g. the average age of all New Zealand residents. Then sample \(\{y_i\}\) (e.g., someone’s age) is not assumed to be instrinsicaly random like in the design-based framework; instead the randomness inherited from the sampling design.

Model-based vs Design Based

Model-based

  1. Define a statistical model

\[y_i = \beta_0 + \beta_1x_i + \epsilon_i\]

  1. Make the \(y_i\)’s random by imposing a distributional assumption

\[\epsilon_i \sim N(0,\sigma^2)\]

  1. “Conditionality Principle”

\[y_i = \beta_0 + \beta_1x_i + \beta_2I_{strata1}+\beta_3I_{strata2} \dots + \epsilon_i\]

\[\epsilon_i \sim N(0,\sigma^2)\]

Design-based

  1. Define the sampling frame, sample design, sampling scheme
  • sample frame: list of primary sampling units \(\{1,\dots, N\}\)
  • sample design: probability (_i) of selection assigned to each unit \(i\)
  • sampling scheme: a draw-by-draw mechanism to sample units according to the probabilities assigned to each unit
  1. Construct estimators for the finite population parameters using inverse probability weighting

\[\widehat{y_{total}} = \sum_{j \in S} \frac{1}{\pi_i} y_j\]

where \(y_{total} = \sum^N y_i\)

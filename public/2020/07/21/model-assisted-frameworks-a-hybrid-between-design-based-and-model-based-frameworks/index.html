Model-assisted frameworks: a hybrid between design-based and model-based frameworks

2020-07-21

To overcome the limitations of both the pure model-based and pure design-based frameworks, a hybrid model-assisted framework has been developed. In this framework a statistical model is specified to determine the parameters of interest but estimation and inference of said parameters is done under a design-based framework.

For example consider a model fit on the finite population of size \(N\) as

\[y_i = \beta_0 + \beta_1x_i\]

then the finite population quantity of interest are \(\beta_0\) and \(\beta_1\) values that minimizes that squared error:

\[\sum_{i = 1}^N (y_i - \beta_0 - \beta_1x_i)^2.\]

Note that similar to other Horvitz-Thompson estimators, this is a simple sum over each individual. Then under a complex sampling design we can construct our estimators of \(\beta_0\) and \(\beta_1\) as the values that minimize

\[\sum_{i \in S}\frac{1}{\pi_i} (y_i - \beta_0 - \beta_1x_i)^2\]

The combination of the two frameworks allows for specifying of a linear model while simultaneously taking into account the sampling design.

We would like to extend this to linear mixed models, which have broad applications.

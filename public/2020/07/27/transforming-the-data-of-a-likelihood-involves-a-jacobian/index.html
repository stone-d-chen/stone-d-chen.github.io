Transforming the data of a likelihood involves a Jacobian; why the likelihood ratio is the key object

2020-07-27

The likelihood ratio is the primary object of interest because if we transform the data \(X\) to \(Y\), our information about a fixed parameter \(\theta\) should not change. This mathematically follows from calculus.

Transforming the data (random variable) of a likelihood involves a Jacobian term

\[L(\theta; y) = L(\theta; x)\left| \frac{\partial x}{\partial y}\right|\]

The [[Classical definition of the likelihood]] starts with a statistical model and thus a probability density \(p_\theta(x)\).

Now assume we perform some 1-1 transformation of \(X\) to \(Y\). Then

\[p_\theta(y) = p_\theta(x(y))*\left| \frac{\partial x}{\partial y}\right|\]

because the definition of the probability density involves an integral: \(P(y(a) \leq Y \leq y(b)) = \int_{y(a)}^{y(b)} p_\theta(y)dy\).

Thus we can directly say that the likelihoods are

\[L(\theta; y) = L(\theta; x)\left| \frac{\partial x}{\partial y}\right|\]

Why the likelihood ratio is the key object

Weâ€™ve stated intuitively that transforming the data \(x\) to \(y\) in a 1-1 manner should not change the information contained in the likelihood. Mathematically we have that transforming the data requires multiplication by the Jacobian.

Then to preserve the intuitive idea, we should examine only the likelihood ratios,

\[\frac{L(\theta_2; y)}{L(\theta_1; y)} = \frac{L(\theta_2; x)}{L(\theta_1; x)},\]

because regardless of what transformation the data is in, the inference between \(\theta_1\) and \(\theta_2\) are the same.

Because of this we say that proportional likelihoods carry the same information. That is \(L(\theta, x)\) carries the same information (better to think about this as shape) that \(c*L(\theta,x)\). As such we typically normalize the curve shape so the value at the max is \(1\).

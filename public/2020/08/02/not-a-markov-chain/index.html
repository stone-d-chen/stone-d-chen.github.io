A not markov chain example

2020-08-02

Showing a stochastic process is not a Markov chain by example

The definition of a Markov chain states that a sequence \((X_n)\) is a MC if the following holds:

\[P(X_{n+1}|X_n, \dots X_0) = P(X_{n+1}|X_n)\]

  • for all \(x\) values 1

That is the probability of what happens in the next step only depends on the previous step. Or put another way, there’s no additional information in \((X_{n-1}, \dots, X_0)\) so “knowing”/conditioning on it does not change the probability of \(P(X_{n+1}|X_n)\).

How do we show something is NOT a Markov chain? We can show that the defining Markov definition doesn’t hold, that is AT LEAST ONE sequence \((X_n,\dots,X_0)\) such that

\[P(X_{n+1}|X_n, \dots, X_0) \neq P(X_{n+1}|X_n)\]

We could show this by directly computing \(P(X_{n+1}|X_n)\) and showing that it’s different for some sequence \((X_n,\dots,X_0)\).

Alternatively we can pick two different sequences \((X_n, \dots,X_0)\) and show \(P(X_{n+1}|\dots)\) gives different probabilities. and that’s the approach we’ll take.

The set-up

Let sequence \((X_n)\) be defined by \(X_n = Q_n + Q_{n-1}\)

So to show that the sequence is NOT a Markov chain, we will construct a situation where knowing \((X_{n-1}, \dots, X_n)\) does change the probability. We will consider the sequences all \(8\)’s and all \(4\)’s because this gives us complete information about what the actual coin flips are.

\(X_n = 6\), the rest are \(8\)’s

Consider the sequence:

  • all \(8\)’s: \(( X_{n-1} = 8, \dots, X_0 = 8)\)

Then fix \(X_n = 6\). This means that \(Q_n = 2, Q_{n-1}=4\) (because we know \(X_{n-1}, \dots, X_n\) we can fully determine \(Q_n\)).

Then \(P(X_{n+1}|X_n = 6, X_{n-1} = 8, \dots, X_0 = 8)\) has distribution:

\[P(X_{n+1}=4|\dots) = P(Q_{n+1}=2) = 1/4 \] \[P(X_{n+1}=5|\dots) = P(Q_{n+1}=3) = 1/2 \] \[P(X_{n+1}=6|\dots) = P(Q_{n+1}=4) = 1/4 \] \[P(X_{n+1}=7|\dots) = P(Q_{n+1}=5) = 0 \]

\(X_n = 6\) but the rest are \(4\)’s

But if \((X_{n-1}=4,\dots,X_0 = 4)\) and then we fix \(X_n = 6\) then we know \(Q_{n} = 4, Q_{n-1} = 2\)

\(P(X_{n+1}|X_n = 6, X_{n-1} = 4, \dots, X_0 =4)\) has distribution

\[P(X_{n+1}=4|\dots) = P(Q_{n+1}=0) = 0 \] \[P(X_{n+1}=5|\dots) = P(Q_{n+1}=1) = 0 \] \[P(X_{n+1}=6|\dots) = P(Q_{n+1}=2) = 1/4 \] \[P(X_{n+1}=7|\dots) = P(Q_{n+1}=3) = 1/2 \] \[P(X_{n+1}=8|\dots) = P(Q_{n+1}=3) = 1/4 \]

The distributions are different

That is we’ve changed the distribution given that we have knowledge other than \(X_n\). The process is NOT memoryless and thus not a Markov chain.

Formally a Markov chain must have

\[P(X_{n+1}|X_n, \dots X_0) = P(X_{n+1}|X_n)\]

which also means more specifically \[P(X_{n+1}|X_n=6) = \] \[P(X_{n+1}|X_n = 6, X_{n-1} = 8, \dots, X_0 =8) = P(X_{n+1}|X_n = 6, X_{n-1} = 4, \dots, X_0 = 4) \]

but we’ve shown that the last equality does not hold.

2020-08-13

Using R via Visual Studio Code or Rstudio server

Two of the more pleasant ways of using R are through Visual Studio Code or install RStudio Server.

RStudio Server

RStudio server will allow you to locally host a RStudio server on WSL and then access it via a webbrowser. You get the full RStudio experience.

  • Rstudio server

Visual Studio Code

The benefits of Visual Studio Code are the easy interop between WSL2 as well as all the extensions. It’s been very easy to access remote R virtual machines and execute code in an poor man’s RStudio way. The easiest way to install is to to just launch vscode from WSL2 using code . to open the current directory. It will the automatically ask to install WSL remote access.

Then you want to install extensions
